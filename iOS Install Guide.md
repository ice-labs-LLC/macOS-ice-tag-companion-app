# HOW TO INSTALL ON iOS
1. Download the app "a-Shell" https://apps.apple.com/us/app/a-shell/id1473805438
2. Download the latest build that ends in "-ios"
3. Click the ZIP. It should make a folder called "YOURVERSION-ios"
4. Run "cd path/to/ios/build/YOURVERSION-ios"
5. Run "swift ContentView.swift"
6. You should have the app!
